Shorten a url(support get method only):
http://${domain}/shorten?url=${url} returns http://${domain}/${code}
* In this site,${domain} is s.jo32.info; ${url} represets the url u wish to shorten.
* e.g. http://s.jo32.info/shorten?url=google.com returns http://s.jo32.info/1

Get a url(support get method only):
http://${domain}/shorten?code=${code}
* ${code} represents the code u get above.
* e.g. http://s.jo32.info/shorten?code=1 returns http://google.com
